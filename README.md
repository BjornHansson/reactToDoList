# reactToDoList

Generated and started the example app:
```
npm install -g create-react-app
create-react-app my-app

cd my-app
```

Start the client:
```
npm start
```

Start the server:
```
node src/server.js
```

Start the db:
```
mongod
```

Jshint:
```
npm run verify
```

For production. The built React app will be served by the node server (http://localhost:3001/):
```
npm run build
node src/server.js
```
